# SRYSO Repertoire Browser

This repository contains the tooling, documentation, and metadata used to build and maintain the SRYSO Repertoire Library.

Repository contents:

- `reference/` — canonical normalization tables
- `scripts/` — survey, migration, and maintenance scripts
- `data/` — repertoire database
- `documents/` — repository architecture and conventions
- `documentation/` — browser and project documentation
- `logs/` — migration surveys and execution history

The actual PDF library is intentionally **not** tracked in Git and resides in Google Drive.
Canonical score and part PDFs are maintained separately in the SRYSO Google Drive repository.

This repository emphasizes canonical organization over historical preservation. Historical source material remains available in the legacy Google Drive archive; this project exists to provide a clean, maintainable repertoire library for ongoing use.
